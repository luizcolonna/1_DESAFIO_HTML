O QUE É O HTML?

 voltar

O HTML,também conhecido como HyperText Markup Language (Linguagem de Marcação de Hipertexto), é uma linguagem de marcação utilizada para criar páginas web. Ele define a estrutura e o conteúdo de uma página web, permitindo que os navegadores entendam como exibir o conteúdo para os usuários.

TAG's

Tags são o conjunto de caracteres que formam um elemento. Existem dois tipos de tags: as que necessitam e as que não necessitam de fechamento.

Para realizar a abertura da tag, utilizamos o sinal de menor ( < ), seguido do nome do elemento e o sinal de maior ( > ).

Para realizar o fechamento, utilizamos o sinal de menor ( < ), seguido de barra ( / ), nome do elemento e o sinal de maior ( > ).

- Tag html

A tag html serve para definir que tudo que estiver dentro dela é ou faz parte de uma página html.

- Tag head

A tag head contém metadados sobre o documento HTML, que não são exibidos no navegador, como por exemplo os links para folhas de estilo (CSS), scripts JavaScript, entre outros.

- Tag title

A tag title é o título que aparece na guia do navegador e nos resultados dos motores de busca.

- Tag body

A tag body contém todo o conteúdo visível de um documento HTML, é basicamente o corpo do site e pode incluir textos, imagens, tabelas, formulários, entre outros.

- Tag strong

A tag strong serve para deixar o texto negrito.

- Tag a

A tag a é usada para criar hiperlinks, permitindo que os usuários cliquem e realizem a navegação para outra página do site.

- Tag img

A tag img é usada para inserir imagens em uma página web.

- Tag i

A tag i serve para deixar o texto itálico.

- Tag input

A tag input é usada para criar campos de entrada de dados, e o tipo de dado a ser inserido pode ser especificado pelo atributo type.

- Tag p

A tag p é usada para definir um parágrafo.

A tag mark destaca um trecho do conteúdo do documento.

- ATRIBUTOS DE TAG's

voltar

Atributos são informações que passamos na Tag para que ela se comporte da maneira esperada. Existem atributos globais (que funcionam em todas as Tags) e específicos (que são direcionados para cada Tag, através de especificação).
